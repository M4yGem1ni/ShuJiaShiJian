# 演示账号

以下账号可用于登录系统体验：

| 角色 | 账号（手机号） | 密码 |
|------|---------------|------|
| 系统管理员 | admin | 123456 |
| 公益组织 | org | 123456 |
| 捐赠者 | donor | 123456 |
| 匿名用户 | anonymous | nopass |

> 手机号字段也支持按用户名匹配登录。

## 后台管理

管理员登录后台：http://localhost:3001/admin/login

## 编译版说明

- 数据库自动创建在运行目录下的 `dev.db`
- 首次运行时若 `dev.db` 不存在会自动初始化并写入演示数据
- 如需重置，删除 `dev.db` 文件后重启即可
